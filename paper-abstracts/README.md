# PaperGym 논문 초록 아카이브

> 목적: PaperGym 6개월 커리큘럼에 들어가는 논문들의 초록을 영어 원문과 한국어 학습용 번역으로 보관한다.

---

## 파일 구조

| 파일 | 내용 |
|---|---|
| `abstracts-en-ko.md` | arXiv에서 수집한 29편 영어 원문 초록. 한국어 번역 자리 포함 |
| `ko-translations/core-abstracts-ko.md` | core 논문 13편 한국어 학습용 초록 번역 |
| `ko-translations/supplemental-abstracts-ko.md` | supplemental 논문 16편 한국어 학습용 초록 번역 |

---

## 주의

- 한국어 번역은 공식 번역이 아니다.
- 공식 기준은 각 논문의 영어 원문 초록과 PDF다.
- 한국어 번역은 PaperGym의 Korean Scaffold Mode에서 학습 보조용으로 사용한다.

---

## 활용 방법

### PaperGym S1 Prescription에서

- 영어 원문 초록은 source preview로 사용
- 한국어 번역은 Korean Scaffold Mode에서 표시
- 3-line Korean Study Hook은 처방전의 “왜 이 논문을 읽는가” 카드로 사용

### 한국어 학습자용 흐름

1. 한국어 3-line hook을 먼저 읽는다.
2. 영어 원문 Abstract를 읽는다.
3. 한국어 학습용 번역을 보며 막힌 문장을 확인한다.
4. 다시 영어 Abstract로 돌아가 핵심 용어를 표시한다.
5. 논문을 덮고 한국어로 3문장 회상한다.
